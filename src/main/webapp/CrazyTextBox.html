CrazyTextBox Demo Project

Shortcuts

  • BACKSPACE - delete symbol that stands before current cursor position
  • CTRL + A - select all symbols
  • CTRL + C - copy selected symbols to buffer
  • CTRL + X - cut selected symbols to buffer
  • CTRL + V - paste symbols from buffer after current cursor position one by one

Implementation simplifications

  • We use font with fixed height and font-family (Couirier)
  • Instead of maintaining visibility of selection area when lose focus (blur), we simply show selected area
  • Cursor positioning works as right from symbol (for mouse-based positioning), so 0 coordinate can be achieved only with keyboard positioning
  • Vertical alignment of flipped chars works correctly only for english chars, special symbols and digital numbers
  • Focus management achieved via extending FocusPanel.

Unimplemented feautures

  • If the text doesn't fit its horizontal bounds, the control must draw the text up to the last fully visible letter and let the user move back and forth to the parts of the text which don't fit, using a cursor.
  • An attempt to select a text beyond the control bounds shall cause the input to scroll (left or right) to show the current cursor position as it moves (left or right) along the course of the selection.
  • Doesn't tested for touch enabled devices (iOS, Anroid)
